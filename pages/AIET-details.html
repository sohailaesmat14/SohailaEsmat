Graduation Project

Pipeline Inspection Robot

Grade: Excellent (A+)

An intelligent robotic system designed to inspect oil pipelines remotely, detect structural faults, and enhance preventive maintenance using sensors and computer vision.

My Role:

  • Designed and implemented the computer vision system to detect defects such as cracks, rust, bends, and leaks.
  • Trained and deployed machine learning models for real-time fault detection.
  • Integrated the Raspberry Pi camera with the detection system.
  • Collaborated on integrating sensor data with the monitoring interface.
